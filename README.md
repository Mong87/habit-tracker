# Habit Tracker

Habit Tracker adalah aplikasi web sederhana untuk mencatat kebiasaan harian. Project ini dibuat menggunakan HTML, CSS, dan JavaScript.

# Fitur
- Menambahkan habit baru
- Menandai habit sebagai selesai
- Menghapus habit
- Menghapus semua habit
- Statistik total, completed, dan remaining
- Progress bar otomatis
- Data tersimpan di localStorage

# Teknologi
- HTML
- CSS
- JavaScript
- localStorage

# Cara Menjalankan
Buka file `index.html` di browser.

# Habit Tracker

Habit Tracker is a simple web application for tracking daily habits. This project is built using HTML, CSS, and JavaScript.

# Features
- Adding new habits
- Marking habits as completed
- Deleting habits
- Deleting all habits
- Total, completed, and remaining statistics
- Automatic progress bar
- Data saved to localStorage

# Technology
- HTML
- CSS
- JavaScript
- localStorage

# How to Run
Open the `index.html` file in a browser.
